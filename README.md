### 01.init

> Proyecto base generado con Next.js utilizando:

- TypeScript
- ESLint
- Tailwind CSS
- App Router (`/app`)
- Soporte para Turbopack

Esta rama contiene el proyecto tal como se genera al correr `npx create-next-app@latest my-project`.

1. Clona el repositorio:

